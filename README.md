> **Creating in-camera visual effects for virtual production. Where creativity begins and reality ends.**

Open Slate is a cutting-edge VFX studio specializing in visual effects, virtual production, 3D assets creation, concept art, and shoot supervision for films, TVC, and OTT platforms.

---

## ✨ Features

- **🚀 Real-time VFX:** Cutting-edge visual effects powered by Unreal Engine and Houdini.
- **🌌 Virtual Production:** In-camera VFX for LED volume stages.
- **🏗️ 3D Asset Creation:** High-fidelity 3D assets and environments.
- **🎭 Lookdev & Lighting:** Real-time look development and cinematic lighting.
- **🎬 Shoot Supervision:** On-set expertise to ensure seamless post-production integration.

---

## 🛠️ Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** CSS Modules, Tailwind CSS
- **3D Graphics:** Three.js, React Three Fiber, React Three Drei
- **Animations:** GSAP (GreenSock Animation Platform), Lucide React Icons
- **Language:** TypeScript

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Akankshashelar96/NextjsAPP.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
